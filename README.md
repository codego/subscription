<b>To start</b>
<p>configure database on .env</p>
```
<p> > composer install</p>
<p> > php artisan optimize</p>
<p> > php artisan migrate</p>
<p> > php artisan db:seed</p>
<p> > php artisan serve</p>
```

Some basic Git commands are:
```
git status
git add
git commit
```
<p>Enjoy...</p>
